# Overview
- Sample project to experiment with NextJS and Sockets

## Structure
- NextJS for frontend
- Express for backend

# Installing
## Windows
- `install.bat`

## Installing frontend individually
- `cd frontend`
- `npm install`

## Installing backend individually
- `cd backend`
- `npm install`

# Running App
## Windows
- `run.bat`

## Running frontend individually
- `cd frontend`
- `npm run dev`
- visit http://localhost:3000

## Running backend individually
- `cd backend`
- `node server.js`
- visit http://localhost:3000
